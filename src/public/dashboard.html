🚀 CI/CD Node.js Application

A fully automated Continuous Integration & Deployment workflow using GitHub Actions & Docker.

📘 Project Overview

This Node.js app showcases a complete CI/CD workflow — from code commit to automated build, test, image creation, and deployment.

  • Express.js Backend
  • Docker Image Build & Push
  • GitHub Actions Pipeline

🧠 Tech Stack

  • ⚙️ Node.js (Express Framework)
  • 🐳 Docker for containerization
  • 💻 GitHub Actions for CI/CD
  • ☁️ AWS EC2 / Docker Hub deployment
🌐 View on GitHub

🔄 CI/CD Pipeline Flow

  1. Developer commits code to GitHub
  2. GitHub Actions triggers build
  3. Docker image built and tagged (v1, v2...)
  4. Image pushed to Docker Hub
  5. App auto-deployed to environment
